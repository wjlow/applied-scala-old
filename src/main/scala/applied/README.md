# Applied Scala

All exercises in this package are related to building a web app.

## To run

```
$ sbt run
```

`sbt` will prompt you to specify which level to run:

```
Multiple main classes detected, select one to run:

 [1] day2.level01.Main
 [2] day2.level02.Main
 [3] day2.level03.Main

Enter number:
```

If working in level03, type: `3`
